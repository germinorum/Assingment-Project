🎉 Beginner's Guide to Installing Windows Server 2019 on VMware 🎉

Follow this fun and easy guide to set up your own Windows Server 2019 VM! 💻

Step-by-Step Installation Guide 🛠️

  1. Step 1: Visit VMware’s Website 🌐

    Right Click on your browser and select "Open Link" to visit the official VMware website.

    👀 On Screen: Look for the Downloads section at the top. Time to grab your VMware software!

  2. Step 2: Download VMware 🖱️

    Once there, Left Click on Downloads, then choose VMware Workstation or VMware Fusion (depending on your operating system).

    👀 On Screen: Select your OS (Windows, macOS, or Linux), then Click Download to start!

  3. Step 3: Get Windows Server 2019 ISO 🌍

    Now go to the Microsoft Evaluation Center and Download the Windows Server 2019 ISO. It's free for evaluation!

    👀 On Screen: You’ll need a Microsoft account—sign in or create one for free!

  4. Step 4: Fire Up VMware 🔥

    Double Click the VMware icon on your desktop to open the program. Let’s get this started!

  5. Step 5: Create Your Virtual Machine 🖥️

    In VMware, Click on **Create a New Virtual Machine** under the File menu to start building your VM!

    👀 On Screen: Choose Custom (Advanced) to tweak your VM settings step-by-step. Hit **Next** to proceed.

  6. Step 6: Select Your ISO File 📂

    When the setup asks for the installation media, Click Browse and choose the Windows Server 2019 ISO file you downloaded earlier.

    👀 On Screen: After selecting, hit **Next** to move forward!

  7. Step 7: Name Your VM & Save 📁

    Give your VM a cool name (like "ServerMaster2019") and choose where you’d like to save the VM files. Then Click Next to continue!

  8. Step 8: Customize Hardware Settings 🔧

    Configure your VM’s hardware:

    • Memory: Allocate 4GB (4096MB).
    • Network Adapter: Set it to **Bridged** for network access.
    • Hard Disk: Allocate at least 40GB for Windows Server.

    Click Finish when you're done configuring.

  9. Step 9: Power Up the VM 🚀

    Click **Power on this virtual machine** to boot up your VM! It will start loading from the ISO file.

  10. Step 10: Install Windows Server 2019 🖥️

    Follow the installation instructions, choosing your preferred language, time zone, and network settings. Just keep clicking **Next** to proceed!

  11. Step 11: Configure the 🧱🔥 Firewall

    After installation, protect your VM with Windows 🔥 Firewall 🧱. Go to Control Panel > System and Security > Windows Defender Firewall** to adjust your settings.

    👀 On Screen: Turn the firewall on/off or tweak advanced settings for extra security!

  12. Step 12: Log In and Explore! 🕹️

    Once installation is complete, log in with your new admin credentials. Congrats! You now have a fully functional Windows Server 2019 VM ready for action!

🎉 Bonus Tips & Tricks 🎉

Here are some fun things to try out:

  • Take a Snapshot: Before experimenting, take a snapshot of your VM to restore it later if needed.
  • Explore Roles: Check out Server Manager and add roles like DNS or Active Directory to level up your server!
